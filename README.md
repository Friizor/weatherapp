## Technologies Used
- Frontend: HTML, CSS
- Backend: JS
- API: https://openweathermap.org/

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Friizor/weatherapp.git
   ```
2. Navigate to the project directory:
`````bash
cd weatherapp
`````
